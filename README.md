Mystery Adventure Game

Mystery Adventure Game is a text-based  adventure where you explore rooms, solve puzzles, and collect items.

Features : 

Explore 5 rooms: FCSE, ACB, CAFE, TUC, LIBRARY.

Solve puzzles in each room to unlock rewards.

Collect items in your inventory (max 10).

Save and load your game progress.

Retry puzzles using recursion.

How to Play

Run the game.

Choose Start New Game or Load Game.

In each room, you can:

View room details

Solve puzzles

View inventory

Move to the next room

Save game

Complete all rooms to finish the adventure
